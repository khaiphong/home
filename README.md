# Home

Home is an injected microservice of KP-Platform to enable each registered [ <b>user / persona / organizaion</b> ] having accesses to the clones of [ <b>facebook / aka Home pages</b> ], [ <b>twitter / aka Mu / or Miss you</b> ], [ <b>linkedin / aka CHR (<a href="https://www.chathamhouse.org/about-us/chatham-house-rule" target="_blank">Chatham-House-Rule</a>) / confidential Hr (Human Resource)</b> ]. The differences are (1) data generated by social apps and their connected value adds by others belong to both the original authors and the root pesona for their AI data mining toward what count, (2) the social apps can be first on the persona private domain, then selected portions can be moved to public domains for wider audiences, (3) versioned available data in consistent distributed databases are available for offline backup from user's <a href="https://github.com/khaiphong/muhub/" target="_blank">MuHub</a> (local networks of Persona IoT).
